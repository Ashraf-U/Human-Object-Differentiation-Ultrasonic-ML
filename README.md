# Human-Object Differentiation Using Vertically Oriented Ultrasonic Sensing and Machine Learning

## Overview
This project focuses on distinguishing humans from objects using vertically oriented ultrasonic sensors combined with machine learning techniques.

## Objectives
- Collect ultrasonic sensor data using vertical orientation
- Extract meaningful features from sensor signals
- Train machine learning models for classification
- Evaluate performance in real-world scenarios

## Technologies Used
- Ultrasonic sensors
- Machine Learning (e.g., SVM, Random Forest, Neural Networks)
